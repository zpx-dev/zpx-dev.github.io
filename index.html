Simple TOML config.
Zero boilerplate.

Define services, routes, and middleware in a single file. SIGHUP to reload without dropping connections.

  • Multiple named TLS and plain listeners
  • Six load balancing algorithms
  • Health checks and circuit breakers
  • ACME certificates with DNS-01 wildcards
  • Docker container auto-discovery
  • Cascading middleware defaults
  • Config includes with glob patterns
zpx.toml 
[[listener]]
name = "https"
bind = "0.0.0.0:443"
compress = true

[[acme]]
name     = "default"
domains  = ["*.example.com"]
dns_provider = "dns_cf"

[[service]]
name     = "api"
backends = [
  "h2c://10.0.0.1:3000",
  "h2c://10.0.0.2:3000",
]
balance = "peak_ewma"

[[route]]
host    = "api.example.com"
service = "api"
rate_limit_rps = 500
retry_attempts = 2

Everything you need.

Production-grade features without the runtime overhead of Go or the complexity of Envoy.

HTTP/2 Frontend
Full h2 over TLS via ALPN with CPU-pinned worker pool and nghttp2 framing.
h2c Upstream
Multiplexed HTTP/2 cleartext to backends with streaming flow control and GOAWAY retry.
kTLS Offload
Cipher keys pushed to kernel after handshake. Encrypt/decrypt in kernel space, zero userspace copies.
Load Balancing
round_robin, least_conn, least_latency, consistent_hash, peak_ewma, p2c.
Health Checks & Circuit Breakers
Automatic backend health polling. Per-backend circuit breaker with configurable threshold and half-open recovery.
ACME Certificates
Automatic provisioning and renewal. HTTP-01 and DNS-01 with 90+ providers. Multi-cert SNI.
Hot Reload
SIGHUP reloads routes, services, pools, rate limiters, allowlists, and TLS certs atomically.
Docker Discovery
Auto-discover containers via Docker socket labels with live event streaming.
TLS Passthrough
Forward raw TLS to backends without terminating. Route by SNI peek.
WebSocket Proxy
Transparent upgrade and bidirectional proxying with origin filtering.
Canary Deployments
Route a percentage of traffic to a canary service. Configurable per-route.
Request Mirroring
Fire-and-forget traffic mirroring with percentage-based sampling.
Compression
Brotli, zstd, and gzip with Accept-Encoding q-value negotiation. Vendored zlib-ng.
Rate Limiting
Per-route token-bucket by client IP. Per-IP connection limits across all listeners.
Forward Auth
Delegate authentication to an external service before proxying.
Prometheus Metrics
Per-service counters, latency histograms, backend health, process and system stats.
Admin Dashboard
Real-time metrics with WebSocket push. Request rate, latency percentiles, backend health.
Smart Routing
Host + path with specificity scoring, weights, listener filtering, POSIX regex, HSTS, CORS.
